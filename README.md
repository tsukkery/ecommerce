# E-Commerce Backend Service

Надежный бэкэнд-сервис для электронной коммерции, созданный на Kotlin и Ktor, включающий аутентификацию пользователей, управление товарами, обработку заказов и административную аналитику.
## 🚀 Функционал

- Регистрация пользователей и аутентификация JWT
- Каталог товаров с кэшированием Redis
- Создание и управление заказами
- Панель администратора для управления товарами и заказами
- Статистика и аналитика заказов
- Асинхронная обработка событий с помощью RabbitMQ
- Подробная документация API с использованием Swagger

## 🛠️ Технологии

- **Kotlin** - Основной язык разработки
- **Ktor** - Веб-фреймворк
- **PostgreSQL** - База данных
- **Exposed** - ORM-фреймворк
- **Redis** - Кэширование
- **RabbitMQ** - Очередь сообщений
- **JWT** - Аутентификация

## 📋 Требования

- JDK 17 or higher
- PostgreSQL (если запускается локально)
- Redis (если запускается локально)
- RabbitMQ (если запускается локально)
